# 声音合并与交叉

## 两个示例
原音频
```
bensound-creativeminds_10s.wav
bensound-summer_10s.wav
```
合并与交叉后的音频
- 两个音频的合并
```
combine_sound.wav
```
- 两个音频的交叉 
```
cross_sound.wav
```
